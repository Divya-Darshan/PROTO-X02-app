proto-x02 Blender Add-on

Load videos, auto-track & render with GloMAP/FFMPEG

About

proto-x02 is a Blender add-on created by Divya Darshan, inspired by PolyFord. It allows you to:

  • Load video into Blender's Video Sequencer
  • Automatically create GloMAP, FFMPEG, and script folder structures
  • Copy templates and dependencies
  • Run auto-tracking via track.bat directly from Blender
  • Cancel tracking anytime

Download & Install

Click the button below to download the add-on ZIP file. Then in Blender: Edit → Preferences → Add-ons → Install, select the ZIP, and enable it.
