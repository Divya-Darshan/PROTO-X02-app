üîç Overview

proto-x02 automates video-to-3D workflows in Blender, including:

  • Project setup & folder creation
  • Auto file placement
  • Video tracking & rendering automation
  • Project zipping via a custom key command

This project is inspired by Poly Ford's videos on photogrammetry pipelines.

‚ú® Features

1. Video Import

  • Supports formats: .mp4, .mov, .avi, .mkv
  • Auto-copies video into 02 VIDEOS
  • Adds video to Blender Sequencer

2. Project Folder Automation

  • Auto-generates folders:
    • 01 GLOMAP
    • 02 VIDEOS
    • 03 FFMPEG
    • 04 SCENES
    • 05 SCRIPT
  • Copies placeholders and templates into folders

3. Auto-Tracking & Rendering

  • Runs track.bat from 05 SCRIPT
  • Tracks video and renders output to 04 SCENES
  • Buttons for start & cancel

4. Cleanup

  • Deletes previous video if removed from Sequencer
  • Keeps project clean

5. Custom Bot Command: I

  • Packages entire project using 7-Zip
  • Creates a shareable archive in project root

üß† How It Works

  1. Open Video: Video added + folders auto-created.
  2. Track: Runs track.bat in terminal.
  3. Cleanup: Removes unused video files.
  4. Package: Press I to zip everything.
